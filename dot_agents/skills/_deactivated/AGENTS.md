# Deactivated Skills

Skills temporarily disabled but preserved for later reactivation.

## Commands

```bash
rei deactivate 'skill-name'  # Move skill here (alias: `rei on`)
rei activate 'skill-name'    # Restore to active (alias: `rei off`)
```

## Why Deactivate?

- Preserves work without deleting
- Easy re-enable without rebuilding
- Safe experimentation
